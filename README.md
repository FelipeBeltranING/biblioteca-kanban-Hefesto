# biblioteca-kanban-Hefesto
Aquí tienes una versión de menos de 350 caracteres:  Repositorio para la planificación de un Sistema de Gestión de Biblioteca Universitaria usando metodología Kanban en GitHub Projects. Incluye organización de tareas con Issues, seguimiento del progreso y aplicación de principios ágiles mediante un tablero visual (To Do, In Progress, Done).
